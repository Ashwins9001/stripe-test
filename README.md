# Motivation #
Doing this project to learn about back-end development and to play around more with a Stripe payments integration.

# Overview #
Project currently shows a placeholder inventories page of three different products with varying inventories, prices. Upon selecting the 'buy' option for any, user gets redirected to sandbox Stripe payment page to complete transaction. 

Project uses following technologies
- Docker
- Stripe payments API
- Flask
- HTML/Jinja2
